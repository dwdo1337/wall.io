First-time Setup
1Install Node.js
Wall.io needs Node.js to run the data backend. If you don't have it yet:
Download Node.js (LTS) and run the installer with default options.
Already have it? Click check below.
2Start the backend
Open the backend folder that came with this extension.
Windows: double-click start-windows.bat
Mac: double-click start-mac.command (right-click the first time, choose Open)
A terminal window will open. Leave it running while you use the extension.
3You're all set
Hover over any wallet address on a supported terminal (Trojan, GMGN, Axiom, Padre, DexScreener and more) to see its average hold time.
